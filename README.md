# color_Scheme_Changer
A lightweight, interactive web application that allows users to instantly change the background color of the page by clicking on predefined color buttons.

🚀 Features
Interactive UI: Seamless background color transitions.

Simple Implementation: Built using standard HTML, CSS, and JavaScript.

Customizable: Easily add or remove colors by editing a single array in the JavaScript file.

Responsive Design: Works across all modern browsers and devices.

🛠️ Tech Stack
HTML5: Structure of the application.

CSS3: Styling for the buttons and layout.

Vanilla JavaScript: Logic for event handling and DOM manipulation.
